Web Browser

A web browser is a software application for retrieving, presenting, and traversing information resources on the World Wide Web. An information resource is identified by a Uniform Resource Identifier (URI/URL) and may be a web page, image, video, or other piece of content. Hyperlinks present in resources enable users easily to navigate their browsers to related resources.

Although browsers are primarily intended to use the World Wide Web, they can also be used to access information provided by web servers in private networks or files in file systems.

History

Origins

The first web browser, called WorldWideWeb, was invented in 1990 by Tim Berners-Lee at CERN. It was later renamed Nexus to avoid confusion with the World Wide Web itself.

The first commonly available web browser with a graphical user interface was Erwise, developed by a team of students at the Helsinki University of Technology. The development was stopped in 1994 due to a lack of funding.

Browser wars

The browser wars saw intense competition for dominance in the web browser marketplace. Microsoft bundled its Internet Explorer with every copy of Windows, leading to a dramatic increase in usage share. By 2002, Internet Explorer had reached over 95% market share.

First Browser War (1995-2001)

Netscape Navigator was the dominant browser throughout the mid-1990s. Microsoft responded by developing Internet Explorer and bundling it with Windows. This led to an antitrust lawsuit against Microsoft, which was eventually settled in 2001.

Second Browser War (2004-2017)

The rise of Firefox (originally Phoenix, then Firebird) and later Google Chrome challenged Internet Explorer's dominance. By 2012, Chrome had overtaken Internet Explorer as the most widely used browser. This era also saw the rise of Opera, Safari, and Edge.

Third Browser War (2017-present)

The modern era is characterized by the dominance of Chromium-based browsers, with Google Chrome, Microsoft Edge, Brave, and others all built on the open-source Chromium project. Firefox remains the only major browser using its own independent engine (Gecko).

Features

Modern web browsers provide a range of features:

  • Tabbed browsing — Multiple pages in a single window
  • Bookmarks — Save and organize favorite pages
  • Extensions — Add functionality through plugins
  • Developer tools — Built-in debugging and profiling
  • Private browsing — Browse without saving history
  • Synchronization — Sync bookmarks, history, and passwords across devices
  • Reading mode — Simplified article view
  • Autofill — Automatically fill form fields
  • Spell check — Check spelling in text fields
  • Translate — Built-in page translation

Rendering engines

A web browser's rendering engine is responsible for displaying web content. The main rendering engines are:

Blink
Used by Chrome, Edge, Opera, Brave, and many others. Forked from WebKit in 2013.
WebKit
Used by Safari and all Apple platforms. The original engine behind Chrome (as WebKit).
Gecko
Used by Firefox and Thunderbird. Developed by Mozilla since 1997.
Goanna
Fork of Gecko used by Pale Moon and Basilisk.

Privacy and security

Web browsers have increasingly focused on privacy and security features:

  • Sandboxing — Isolates browser processes from the operating system
  • Same-origin policy — Prevents malicious scripts from accessing data from other sites
  • HTTPS enforcement — Warns users on insecure connections
  • Content blocking — Built-in ad and tracker blocking
  • Cookie controls — Granular control over third-party cookies
  • Fingerprinting protection — Reduces browser fingerprint uniqueness
